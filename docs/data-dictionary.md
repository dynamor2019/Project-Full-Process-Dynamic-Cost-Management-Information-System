# 数据字典

所有模块开发前必须先补充数据字典。字段名称、类型和业务含义经教师确认后方可修改公共数据结构。

| 表/文件 | 字段名 | 中文名称 | 类型 | 必填 | 示例 | 来源 | 业务说明 | 负责人 |
|---|---|---|---|---|---|---|---|---|
| project | project_id | 项目编号 | string | 是 | P001 | 教师数据 | 项目唯一标识 | 教师 |

## 规则

- 金额统一注明含税或不含税及单位。
- 日期统一使用 `YYYY-MM-DD`。
- 比例统一说明使用小数或百分数。
- 缺失值、重复值和异常值必须定义处理方式。
- 禁止把真实姓名、手机号或涉密项目数据放入示例数据。
