# 基座框架约束

## 总原则

本项目是统一整合平台。学生只负责在教师分配的任务范围内补充内容，不负责修改框架。

## 必须统一的内容

- 统一数据库
- 统一 TypeScript 代码结构
- 统一全局 CSS
- 统一导航
- 统一页面标题区
- 统一表单索引 `form_index`
- 统一附件与日志机制

## 学生允许做的事情

- 在指定任务对应的业务页面中增加字段展示
- 增加表单输入项
- 实现数据读取
- 实现数据新增
- 实现数据修改
- 实现数据筛选
- 实现本任务对应的业务规则
- 将本任务记录写入统一索引

## 学生禁止做的事情

- 修改公共导航结构
- 修改全局样式入口
- 修改公共数据库连接方式
- 绕开统一数据库新建独立数据源
- 删除公共索引逻辑
- 改动他组任务目录
- 自行更换技术栈

## 教师与集成组负责的内容

- 公共数据库结构
- 数据库连接和访问封装
- 公共查询条件
- 公共页面框架
- 全局样式
- 类型定义
- 公共工具函数
- 统一看板与综合展示
