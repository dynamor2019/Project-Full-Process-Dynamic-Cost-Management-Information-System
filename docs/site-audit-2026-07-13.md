## 进入学生开发前的阻断项

1. 根目录 `package.json` 引用了尚不存在的 `backend/`、`scripts/setup-db.js`、`scripts/dev.js` 和 `scripts/test-all.js`，因此 README 规划的统一 npm 命令当前不能完整运行。
2. 根目录和前端均没有 `package-lock.json`，不能保证 90 名学生安装出一致依赖版本。
3. 当前展示页主要由 `frontend/app.js` 静态生成，规划中的 React 模块加载入口和 Express 后端尚未接通。
4. `frontend/task-guides.js` 当前只包含 TP-01 的引导内容，其余 24 个任务仍需按同一结构补齐。
5. 当前登录是课程演示登录，不具备真实身份认证、权限控制或服务器会话。
6. 图标脚本来自外部 CDN；断网时文字导航仍可使用，但图标可能不显示。

## 发布判断

当前版本可以作为“课程方案和界面骨架”发布到 GitHub，也可以让学生 Clone 完成环境验收；在统一 npm 命令、后端、模块扩展入口和锁文件完成前，不应开放 25 个正式业务开发任务。
