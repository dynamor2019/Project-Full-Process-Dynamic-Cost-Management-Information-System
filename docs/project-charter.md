# 项目章程

## 项目名称

工程项目全过程动态造价管理信息系统

## 项目目标

围绕工程项目成本、计量支付、变更签证和合同风险，建立统一的数据、预警与报告系统，为造价人员提供可追溯的管理辅助。

## 目标用户

- 建设单位造价管理人员
- 施工单位商务人员
- 工程造价咨询人员
- 项目管理人员

## V0.1 范围

- 支持教师提供的脱敏示例数据。
- 至少完成一个端到端管理模块。
- 业务规则具有来源和人工标准答案。
- 具备本地运行、测试、导入导出和使用说明。

## 暂不实施

- 生产环境部署
- 复杂多级权限
- 电子签章和消息推送
- 未经批准的外部大模型或第三方业务接口
- 真实企业涉密数据

## 项目治理

- 教师：项目负责人、产品负责人和最终验收人。
- 集成组：维护公共底座、模块接口和发布版本。
- 模块组：按 Issue 完成开发、测试和文档。
- `main`：受保护的可运行主分支。

## 版本路线

- `v0.1`：工程管理信息系统课程交付。
- `v0.2-plan`：创新创业基础课程产品方案。
- `v1.0`：创新创业实践课程最终交付。
